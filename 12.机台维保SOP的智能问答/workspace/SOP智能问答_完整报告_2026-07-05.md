## 一、解题背景

作为生产EM部门的新人，工作和SOP、Checklist分不开。传统模式需要手动翻阅SOP文档查找操作步骤，效率低且容易遗漏安全关键项。本系统基于SOP知识库，实现自然语言提问→精确SOP步骤检索→结构化回答的智能问答链路。

### 用户画像

| 维度 | 说明 |
|------|------|
| 角色 | 生产EM部门新人 |
| 痛点 | SOP文档长、步骤多、安全项易遗漏、标准值记不住 |
| 场景 | PM前确认步骤、操作中查询标准值、完工后确认验证项目 |

### Query 1: PM保养前需要做哪些安全准备?

**检索结果**: §1 安全准备 | 相关度: 99% | 匹配: exact:安全

> ⚠️ **SOP原文**:

1.1 确认机台状态为 IDLE，按下 EMO 按钮
1.2 穿戴必要 PPE: 防酸手套、护目镜、防静电鞋、安全帽
1.3 确认腔体温度已降至室温 (<40°C)，检查温度传感器读数
1.4 关闭工艺气体供给: 依次关闭 CF4, CHF3, O2, Ar 气瓶主阀
1.5 确认腔体已通风至少 15 分钟，有害气体探测器读数 < 5ppm
1.6 挂上 "保养中，禁止操作" 警示牌

| 提取标准值 | 数值 | 步骤 |
|-----------|------|------|
| 腔体温度 | < 40°C | 1.3 |
| 有害气体 | < 5ppm | 1.5 |
| 通风时间 | ≥ 15分钟 | 1.5 |

| 提取安全项 | 类型 |
|-----------|------|
| EMO 按钮 | 紧急停机 |
| PPE (防酸手套/护目镜/防静电鞋/安全帽) | 个人防护 |
| 气体供给关闭 | 危险源隔离 |
| 通风15分钟 | 有害气体清除 |
| 警示牌 | 操作禁止标识 |

### Query 2: 上电极(Shower Head)的清洁步骤是什么?

**检索结果**: §2 腔体清洁 | 相关度: 99% | 匹配: exact:上电极

> **SOP原文**:

**2.1 拆卸上电极 (Shower Head):**
- 2.1.1 用扭矩扳手按对角线顺序松开 12 颗 M8 螺栓
- 2.1.2 使用专用吊具将上电极缓慢吊起，放置于清洁台
- 2.1.3 检查上电极表面: 记录腐蚀程度、沉积物颜色和厚度

**2.2 清洁上电极:**
- 2.2.1 用无尘布蘸 IPA 擦拭表面沉积物
- 2.2.2 顽固沉积物使用专用清洁剂 (Clean-100)，浸泡 10 分钟后擦拭
- 2.2.3 用 DI Water 冲洗 3 次，确认无残留
- 2.2.4 用 N2 吹干，在强光下检查表面洁净度

**2.3 清洁腔体内壁:**
- 2.3.1 用真空吸尘器清除腔内松散的聚合物颗粒
- 2.3.2 用 Scotch-Brite 垫 + IPA 擦拭腔壁，特别注意气体注入孔周围
- 2.3.3 检查腔壁 Y2O3 涂层的完整性: 记录剥落/裂纹位置和面积

### Query 3: Focus Ring更换时需要注意什么标准值?

**检索结果**: §3 消耗件更换 | 相关度: 99% | 匹配: exact:更换

> **SOP原文**:

3.1 更换 Focus Ring:
- 3.1.1 记录旧 Focus Ring 的 RF hours (从 PM log 读取)
- 3.1.2 检查旧环: 测量厚度(**目标 > 2.0mm**), 记录磨损量
- 3.1.3 安装新 Focus Ring: 对准定位销，轻压至卡入到位
- 3.1.4 用塞尺检查间隙: **四周间隙应 < 0.1mm**

3.2 更换 Edge Ring:
- 3.2.1 拆卸旧 Edge Ring，检查是否有裂纹或缺角
- 3.2.2 清洁安装槽: 用 IPA + 无尘布清除沉积物
- 3.2.3 安装新 Edge Ring，确认完全贴合

3.3 更换 O-rings (腔门、电极、气体管路):
- 3.3.1 用 O-ring pick 小心取出旧 O-ring，**避免划伤密封槽**
- 3.3.2 检查密封槽: 无划痕、无颗粒
- 3.3.3 新 O-ring 涂薄层真空脂 (Krytox)，均匀涂抹
- 3.3.4 安装新 O-ring，**确保无扭曲、完全入槽**

| 关键标准值 | 数值 |
|-----------|------|
| Focus Ring 厚度 | > 2.0mm |
| Focus Ring 安装间隙 | < 0.1mm |
| RF hours | 从PM log读取并归零 |

| 注意事项 | 风险 |
|----------|------|
| O-ring pick 避免划伤密封槽 | 密封失效→漏率超标 |
| O-ring 无扭曲、完全入槽 | 真空泄漏 |
| 真空脂均匀涂抹 | 密封不均匀 |

### Query 4: ESC静电卡盘检查包含哪些测试项目?

**检索结果**: §4 部件检查 | 相关度: 99% | 匹配: exact:静电卡盘

> **SOP原文**:

4.1 检查 ESC (静电卡盘):
- 4.1.1 测量 ESC 表面温度均匀性: 用热像仪扫描，**温差 < 2°C**
- 4.1.2 检查 He 背冷管路: 无泄漏 (**氦检漏仪 < 1×10⁻⁹ mbar·L/s**)
- 4.1.3 测量 ESC 电阻: **DCE 对地 > 10MΩ**

4.2 检查 RF 系统:
- 4.2.1 检查 RF 馈入线: 无氧化、无松动
- 4.2.2 测量 VSWR (驻波比): 应 **< 1.5** (网络分析仪)
- 4.2.3 检查匹配网络: 电容无鼓包、无异味

4.3 检查真空系统:
- 4.3.1 检查 Gate Valve O-ring: 无颗粒、无变形
- 4.3.2 检查 TMP (涡轮分子泵): **运行电流 < 标称值 110%**
- 4.3.3 检查 Dry Pump: 油位正常、无异响
- 4.3.4 真空漏率测试: 关闭阀门后 **压力上升 < 2mTorr/min**

| 测试项目 | 工具 | 标准值 |
|----------|------|--------|
| ESC 温度均匀性 | 热像仪 | 温差 < 2°C |
| He 背冷泄漏 | 氦检漏仪 | < 1×10⁻⁹ mbar·L/s |
| ESC 电阻 | 万用表 | DCE对地 > 10MΩ |
| RF 馈入线状态 | 目检 | 无氧化/无松动 |
| VSWR | 网络分析仪 | < 1.5 |
| 匹配网络 | 目检+嗅觉 | 电容无鼓包/无异味 |
