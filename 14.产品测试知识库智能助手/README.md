# 产品测试知识库智能助手

PTE 部门新人学习工具 — CP/FT/SLT 测试知识库 + AI 问答 + 术语解析

## 快速开始

```bash
pip install -r requirements.txt
python main.py -q "CP测试流程是什么?"
python main.py -t "RDBI"
```

## 能力

1. **知识问答**: 自然语言提问 → 结构化测试知识回答
2. **术语解析**: 输入术语 → 定义/阶段/关联参数/场景

## 知识库

4 份文档, 基于联网搜索 + LLM 结构化分解:
- CP 晶圆测试 (流程/参数/设备/术语)
- FT 封装测试
- SLT 系统级测试
- 测试术语词典 (11+ 术语)
