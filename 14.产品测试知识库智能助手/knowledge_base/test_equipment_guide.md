# 半导体测试设备速查知识库（PTE新人版）

## 1. 测试流程概述
半导体测试主要分为晶圆针测（CP）、封装后终测（FT）与系统级测试（SLT）三大阶段。不同阶段使用的设备组合差异很大，新人应首先熟悉主流机型及其适用领域。

## 3. 设备选型快速考虑因素
- **测试覆盖率**：能否覆盖规格书所有参数（DC/AC/功能/性能），V93000和UltraFLEX在高速数字与射频上覆盖更广。
- **成本**：含设备折旧、Socket/探针卡耗材、Dock板、场地、维护等。J750往往单芯片测试成本更低。
- **UPH（Unit Per Hour）**：决定产能。Cohu Delta Matrix适合小封装高产器件；多site并测可大幅提升UPH。
- **并行度（Site数）**：同测site越多，UPH越高，但装机复杂度和资源冲突风险也随之上升。SLT中AEM AMPS支持超320站点。
- **温度要求**：CP多为常/高低温，FT需Handler支持三温（-55℃~150℃），SLT常配液冷。


## 4. PTE新人须知：缩写与快速上手

### 4.1 常见设备与岗位缩写
- **PTE**：Product Test Engineer，产品测试工程师
- **CP**：Chip Probe，晶圆针测
- **FT**：Final Test，封装后终测
- **SLT**：System Level Test，系统级测试（在类真实系统环境下测试）
- **ATE**：Automatic Test Equipment，自动化测试机（如V93000、UltraFLEX）
- **DUT**：Device Under Test，被测器件
- **Handler**：机械手，自动取放DUT进行FT的设备
- **Prober**：探针台，自动移动晶圆使探针与pad接触
- **PIB**：Probe Interface Board，探针卡接口板（连接ATE与探针卡）
- **Load Board / DUT Board**：负载板/测试板，安装Socket，连接ATE与DUT
- **UPH**：Unit Per Hour，每小时测试产能
- **Socket**：测试座，FT时放置芯片的精密连接器
- **Probe Card**：探针卡，CP时与晶圆pad接触的针阵

### 4.2 新人如何快速上手
1. **掌握流程**：了解一颗芯片从晶圆到出货的完整测试路径（CP→FT→SLT），以及所使用的设备组合。
2. **死磕一台主流ATE**：优先学习实验室或产线上配置最多的平台（如V93000或UltraFLEX），从基本语法、Pattern调试和Testflow开发开始。
3. **跟线调试**：参与至少一次完整的CP或FT setup：Load Board安装、Socket/探针卡更换、设备自检、开机跑第一个程序、排查报警。这是成长最快的环节。
4. **学会读数据**：看懂STDF/ATDF数据日志，运用数据分析软件（如Exensio、Galaxy等）分析良率和测试项分布。
5. **认识设备与耗材**：现场认清楚Handler、Prober、Load Board、Socket的外观和型号，避免沟通时说不清“什么设备上的什么板子”。
6. **善用内部资源**：查阅设备厂商手册、内部培训视频和前辈的测试程序注释，带着问题去请教会更有收获。
7. **关键思维**：永远把“测试覆盖率”和“成本/UPH”平衡放在首位，这是PTE的价值所在。
